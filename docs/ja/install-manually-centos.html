Install Manually on CentOS

Install requirements

Vulsセットアップに必要な以下のソフトウェアをインストールする

  • SQLite3, MySQL, PostgreSQL, Redis
  • git
  • gcc
  • GNU Make
  • go v1.8.3 or later (The latest version is recommended)
    • https://golang.org/doc/install
$ ssh centos@52.100.100.100  -i ~/.ssh/private.pem
$ sudo yum -y install sqlite git gcc make wget
$ wget https://storage.googleapis.com/golang/go1.8.3.linux-amd64.tar.gz
$ sudo tar -C /usr/local -xzf go1.8.3.linux-amd64.tar.gz
$ mkdir $HOME/go

/etc/profile.d/goenv.sh に以下の行を追加する。

export GOROOT=/usr/local/go
export GOPATH=$HOME/go
export PATH=$PATH:$GOROOT/bin:$GOPATH/bin

カレントシェルに以下の環境変数をセットする。

$ source /etc/profile.d/goenv.sh

Deploy go-cve-dictionary

go-cve-dictionary

$ sudo mkdir /var/log/vuls
$ sudo chown centos /var/log/vuls
$ sudo chmod 700 /var/log/vuls
$
$ mkdir -p $GOPATH/src/github.com/kotakanbe
$ cd $GOPATH/src/github.com/kotakanbe
$ git clone https://github.com/kotakanbe/go-cve-dictionary.git
$ cd go-cve-dictionary
$ make install

$GOPATH/bin以下にバイナリが生成される。

次に、NVDから脆弱性データベースを取得する。
AWS上で実行した場合だと大体10分くらいかかるだろう。

$ cd $HOME
$ for i in `seq 2002 $(date +"%Y")`; do go-cve-dictionary fetchnvd -years $i; done
... snip ...
$ ls -alh cve.sqlite3

-rw-r--r--. 1 centos centos  32K Aug  6 08:10 cve.sqlite3-shm

Deploy goval-dictionary

goval-dictionary

$ mkdir -p $GOPATH/src/github.com/kotakanbe
$ cd $GOPATH/src/github.com/kotakanbe
$ git clone https://github.com/kotakanbe/goval-dictionary.git
$ cd goval-dictionary
$ make install

$GOPATH/bin以下にバイナリが生成される。

次に、RedHatのOVALデータベースを取得する。CentOSからはOVALが提供されていないのでRedHatのものを使う。 README

$ goval-dictionary fetch-redhat 7

CentOS以外をスキャンした場合は、OS、バージョンに対応するOVAL情報を事前に取得する。

  • Alpine
  • RedHat, CentOS
  • Debian
  • Ubuntu
  • Oracle Linux
  • SUSE

Deploy Vuls

$ mkdir -p $GOPATH/src/github.com/future-architect
$ cd $GOPATH/src/github.com/future-architect
$ git clone https://github.com/future-architect/vuls.git
$ cd vuls
$ make install

もしすでにVulsをインストール済みでアップデートする場合は以下を実行する。

$ rm -rf $GOPATH/pkg/linux_amd64/github.com/future-architect/vuls/
$ rm -rf $GOPATH/src/github.com/future-architect/vuls/
$ cd $GOPATH/src/github.com/future-architect
$ git clone https://github.com/future-architect/vuls.git
$ cd vuls
$ make install

$GOPATH/bin以下にバイナリが生成される。
