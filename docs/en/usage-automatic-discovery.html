Automatic Discovery

Discovery subcommand discovers active servers specified in CIDR range, then display the template of config file (TOML format) to terminal.

$ vuls discover -help
discover:
        discover 192.168.0.0/24

Example

$ vuls discover 192.168.11.0/24
