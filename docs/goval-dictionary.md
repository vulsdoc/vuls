## Usage: goval-dictionary on different server

```
$ goval-dictionary server -bind=192.168.10.1 -port=1324
```

Sepcify URL of the goval-dict in config.toml

```
[ovalDict]
url = "http://192.168.0.1:1324"
```

```
$ vuls report 
```
